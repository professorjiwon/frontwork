폼 관련 태그

form태그내에 submit버튼 클릭시 해당 form안에 작성된
사용자가 입력한 값들을 서버로 전송하면서 요청하는 역할 수행

from action="" method=""

* action속성 : 해당 form에 사용자가 입력한 값들을 전달하면서 요청할 서버의 경로 넣어줌 * method속성 : 요청 전송 방식을 지정하는 속성(get(기본값) | post)
> get방식 : 요청시 사용자가 입력한 값들이 url에 노출되는 방식
> post방식 : 요청시 사용자가 입력한 값들이 url에 노출되는 않는 방식
form내의 submit버튼 클릭시 action 지정한 서버로 사용자 입력한 값들이
넘어가면서 요청됨(사용자가 입력한 값들은 key=value 세트로 전달)

첨부파일 :

type="hidden"

- 특정값을 화면에 노출시키지 않고 서버로 전송해야하는 정보가 있을 때 사용


type="submit|reset|button"

- submit : 서버로 form안의 들어있는 모든 값들을 전송 - reset : form안의 값들을 초기화 - button : 일반버튼(사용자 정의 버튼)


button[type="submit|reset|button"]
