CSS 기본선택자

선택자란?

특정 html요소를 선택하고자 할 때 사용하는 기능
해당 요소를 선택하여 원하는 '스타일'과 '기능'을 적용시킬 수 있음

1. 모든(전체) 선택자 : *

현재 이 문서의 모든 요소들을 선택하고자 할 때 사용

        * {
            스타일속성 : 값;
            스타일속성 : 값;
        }

2. 태그 선택자 : 태그명

현재 이 문서의 해당 태그들을 모두 선택하고자 할 때 사용
여러 태그가 동일한 스타일을 갖는가면 컴마(,)로 여러개를 넣을 수 있다 

        태그명 {
            스타일속성 : 값;
        }
        태그명, 태그명 {
            스타일속성 : 값;
        }

3. 아이디 선택자 : #아이디명

현재 문서에서 특정 요소 "하나만"을 선택하고자 할 때 사용
단, 해당 요소에 id속성을 이용하여 고유한 아이디값을 부여해야함 

        #아이디명 {
            스타일속성 : 값;
        }

4. 클래스 선택자 : .클래스명

문서내에서 내가 원하는 요소 "여러개" 선택하고자 할 때 사용

        .클래스명 {
            스타일속성 : 값;
        }
