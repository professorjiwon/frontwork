함수

함수 선언하는 방법

  1. 선언적 함수(Function Declaration) : funcution키워드, 함수명
  2. 익명 함수(Function Expression) : 변수에 익명함수를 대입하는 방식
  3. 화살표 함수(Arrow Function) : 익명함수를 간단히 작성하는 방법(ES6(ES2015)에 추가)
  4. 즉시 실행 함수 iife(Immediately Invoked Function Expression)
    • 선언과 호출
    • 변수를 캡슐화처리 유용

1) 선언적 함수

        function 함수명([매개변수, 매개변수, ...]) {
            해당 함수 호출시 실행할 코드;
            [return 결과값;]
        }

2) 익명 함수

        변수|속성 = function([매개변수,매개변수,...]) {
            해당 함수 호출시 실행할 코드;
            [return 결과값;]
        }

        * 주로 이벤트 핸들러 작성시 사용

3) 화살표 함수

        - 본문에 리턴만 있을 때
            변수 = ([매개변수, 매개변수, ...]) => 리턴값

        - 본문이 여러줄 일 때
            변수 = ([매개변수, 매개변수, ...]) => {
                해당 함수 호출시 실행할 코드;
                [return 결과값;]
            }

4) 즉시실행 함수

        - 정의하면서 즉시 실행되는 함수
        - 단 한번만 호출됨
          ([변수] = function([매개변수, 매개변수, ...]) {
            해당 함수 호출시 실행할 코드;
          })(인자,인자,...);

함수의 매개변수


* 매개변수 관련 arguments배열

함수 호출시 전달되는 값은 내부적으로 arguments 배열에 담김
* arguments : 모든 함수마다 내부적으로 하나씩 존재하는 배열형식 객체

* 매개변수에 this관련 전달하기

this : 현재 이벤트가 발생한 요소 객체

함수의 리턴

* 일반적인 값

* 익명함수를 리턴

이름 :

화살표 함수의 매개변수와 리턴

* 매개변수 지정 방법

    * 매개변수가 1개인 경우를 제외하고는 반드시 소괄호를 넣어야 됨
      - 매개변수가 없을 경우 :        () => return값;
      - 매개변수가 1개인 경우 :       x => return값;  (소괄호 생략가능)
      - 매개변수가 2개 이상인 경우 :  (x, y) => return값;

* 함수 body 지정 방법

    * single line block
        x => { return 리턴값; }
        x => 리턴값, 중괄호 생략
    
    * multi line block
        x => {
            실행구문;
            return 값;
        }
