Content (innerHTML, innerText) 관련 메소드

* html(['문구']) 메소드 : innerHTML 속성과 관련된 메소드

선택된 요소의 content영역의 값을 리턴해주거나 또는 변경해주는 메서드
> $('선택자').html() : 선택된 요소의 content영역 값을 반환해줌(html태그 포함) > $('선택자').html('문구') : 선택된 요소의 content영역 값을 해당 문구로 변경해 줌(html태그가 존재하면 태그로써 해석함)

네이버로

* text(['문구']) 메소드 : innerText 속성과 관련된 메소드

선택된 요소의 content영역의 값을 리턴해주거나 또는 변경해주는 메서드
> $('선택자').text() : 선택된 요소의 content 영역의 값을 반환(html태는 포함x, 오로지 text만) > $('선택자').text('문구') : 선택된 요소의 content 영역의 값을 제시한 문구로 변경(html태그가 존재할 경우 태그해석x)
